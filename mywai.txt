.SH NOME
mywai - mostra informações do usuário

.SH USO
whoami [-ugiG]

.SH DESCRIÇÃO
Mostra informações do usuário associado com o atual userID.

.SH OPÇÕES
.IP -u
mostra o nome do usuário atual

.BR
.IP -g
mostra o nome do grupo atual

.BR
.IP -i
em conjunto com [-ug], mostra o ID do usuário/grupo

.BR
.IP -G
mostra todos os grupos do usuário atual

.BR
